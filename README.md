# Ehky Chat App 💬

A modern, visually appealing chat application built with **Flutter** and powered by **Firebase**. Ehky Chat App provides secure authentication, real-time messaging, and advanced features designed to deliver a seamless user experience.

---

## 🌟 Key Features

### 🎨 Material 3 Design & Flutter Fundamentals
- **Stunning UI/UX**: Designed using the latest **Material 3** principles for a modern, beautiful, and highly responsive user interface.
- **Flutter Best Practices**: Demonstrates solid understanding and leverage of Flutter's capabilities to build performant cross-platform applications.

### 🔥 Firebase Integration
- **Secure Authentication**: Robust user sign-up and login utilizing Firebase Authentication to keep your users' data secure.
- **Real-Time Data Sync**: Powered by Cloud Firestore to ensure messages are delivered and synchronized instantly.
- **Cloud Storage**: Seamless integration with Firebase Storage to handle multimedia assets securely.

### ⚡ Advanced Functionality
- **Dynamic User Experience**: Real-time "Last Seen" and "Online" states to keep chats feeling alive and interactive.
- **Push Notifications**: Integrated with Firebase Cloud Messaging (FCM) to keep users engaged, informed, and always up-to-date with new messages.
- **Multimedia Sharing**: Allow users to share images directly within their chat conversations, making communication richer and more expressive.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **State Management**: `flutter_bloc`
- **Backend**: Firebase (Auth, Firestore, Cloud Messaging, Storage)
- **Design System**: Material 3

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (`^3.11.4` or compatible)
- Dart SDK
- A Firebase project configured for Android/iOS

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ehky_chat_app.git
   cd ehky_chat_app
   ```

2. **Install Dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Follow the [Firebase Flutter Setup Guide](https://firebase.google.com/docs/flutter/setup) to add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS).

4. **Run the App**
   ```bash
   flutter run
   ```
